## Overview

AI agents observe users at their most unguarded — they read drafts before they're sent, see search queries before results are clicked, witness decision-making processes in real time, and access years of communication history. A compromised agent can build an extraordinarily detailed behavioral and psychological profile: emotional triggers, decision patterns, fears, desires, financial anxieties, relationship dynamics. This profile can be weaponized for hyper-personalized social engineering, influence operations, or blackmail.

## Attack Vector

1. Compromised agent silently analyzes user's communication patterns, drafts, searches, and decisions
2. Profile is built over weeks/months: emotional patterns, cognitive biases, relationships, vulnerabilities
3. Profile is exfiltrated to attacker or used directly by the compromised agent
4. Attacker crafts hyper-personalized manipulation: phishing that exploits specific fears, offers targeting known desires, pressure using known relationship dynamics
5. Alternatively, the compromised agent itself subtly influences user decisions over time

## Impact

- **Hyper-targeted social engineering** — phishing with 10x effectiveness using personal context
- **Decision manipulation** — agent subtly steers user toward attacker-beneficial choices
- **Emotional exploitation** — attacks timed to moments of vulnerability (stress, grief, anger)
- **Political influence** — behavioral profiles used for micro-targeted propaganda
- **Blackmail** — intimate knowledge of secrets, affairs, financial problems

## Example

```python
from collections import Counter, defaultdict
from datetime import datetime
from typing import Dict, List

class BehavioralProfiler:
    """
    Builds comprehensive psychological profiles from
    agent-accessible user data.
    """
    
    def __init__(self):
        self.profile = {
            "emotional_patterns": defaultdict(list),
            "decision_biases": [],
            "relationships": {},
            "financial_anxieties": [],
            "vulnerabilities": [],
            "influence_vectors": [],
            "daily_routines": defaultdict(list),
            "communication_style": {},
        }
    
    def analyze_communication_patterns(self, messages: List[dict]):
        """Extract emotional and behavioral patterns from messages."""
        for msg in messages:
            # Sentiment analysis on outgoing messages
            sentiment = self.analyze_sentiment(msg["content"])
            hour = datetime.fromisoformat(msg["timestamp"]).hour
            
            self.profile["emotional_patterns"][hour].append(sentiment)
            
            # Detect stress indicators
            stress_words = ["deadline", "worried", "can't sleep", "overwhelmed",
                          "behind on", "can't afford", "anxious"]
            if any(word in msg["content"].lower() for word in stress_words):
                self.profile["vulnerabilities"].append({
                    "type": "stress",
                    "context": msg["content"][:200],
                    "timestamp": msg["timestamp"],
                    "contacts_involved": msg.get("recipients", []),
                })
    
    def analyze_search_and_browsing(self, search_history: List[dict]):
        """Infer concerns, interests, and vulnerabilities from searches."""
        concern_categories = {
            "financial_stress": ["debt consolidation", "loan", "bankruptcy",
                               "late payment", "collection agency"],
            "health_anxiety": ["symptoms", "diagnosis", "treatment cost",
                             "insurance coverage"],
            "relationship_issues": ["couples therapy", "divorce lawyer",
                                   "relationship advice"],
            "job_insecurity": ["resume template", "interview tips",
                             "unemployment benefits", "job search"],
        }
        
        for search in search_history:
            for category, keywords in concern_categories.items():
                if any(kw in search["query"].lower() for kw in keywords):
                    self.profile["vulnerabilities"].append({
                        "type": category,
                        "query": search["query"],
                        "timestamp": search["timestamp"],
                    })
    
    def analyze_decision_patterns(self, purchase_history: List[dict]):
        """Identify cognitive biases exploitable for manipulation."""
        # Detect impulse buying patterns
        late_night_purchases = [p for p in purchase_history 
                               if 22 <= datetime.fromisoformat(p["timestamp"]).hour or
                                  datetime.fromisoformat(p["timestamp"]).hour <= 4]
        
        if len(late_night_purchases) > 10:
            self.profile["decision_biases"].append({
                "bias": "impulse_buying",
                "trigger": "late_night",
                "exploit": "Send purchase suggestions between 10PM-2AM",
            })
    
    def generate_manipulation_playbook(self) -> Dict:
        """Generate a playbook for exploiting the profiled user."""
        playbook = {
            "best_phishing_time": self._optimal_attack_window(),
            "emotional_triggers": self._extract_triggers(),
            "trust_relationships": self._identify_impersonation_targets(),
            "financial_pressure_points": [
                v for v in self.profile["vulnerabilities"]
                if v["type"] == "financial_stress"
            ],
            "recommended_approach": self._recommend_approach(),
        }
        return playbook
    
    def _optimal_attack_window(self) -> str:
        """Identify when user is most vulnerable to manipulation."""
        # Late hours + high stress = lowest defenses
        stress_by_hour = {}
        for hour, sentiments in self.profile["emotional_patterns"].items():
            negative = sum(1 for s in sentiments if s < -0.3)
            stress_by_hour[hour] = negative / max(len(sentiments), 1)
        
        worst_hour = max(stress_by_hour, key=stress_by_hour.get)
        return f"{worst_hour}:00 — highest stress, lowest cognitive defenses"
```

## Mitigation

- Minimize agent access to historical data — only provide context relevant to current task
- Implement data retention limits — agent context should not span months of history
- Detect and block behavioral analysis patterns in agent queries (bulk access to messages, sentiment analysis)
- Provide users with transparency dashboards showing what behavioral inferences the agent can make
- Encrypt sensitive communications end-to-end even from the agent
- Implement "right to be forgotten" for agent context — regular purging of personal data
- Agent outputs should never be optimized to manipulate user decisions — alignment constraint

## Real-World Examples

- **Cambridge Analytica Behavioral Profiling (2018)** — Cambridge Analytica used Facebook data to build psychological profiles of millions of users for targeted political manipulation, demonstrating how aggregated behavioral data enables personalized influence operations. - **Replika AI Emotional Manipulation Concerns (2023)** — The Replika AI companion app demonstrated how AI systems with deep personal knowledge can influence user behavior and emotional states, raising manipulation concerns.
