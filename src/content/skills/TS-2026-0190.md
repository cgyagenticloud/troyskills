## Overview

AI agents with access to communication tools (email, social media, messaging) and content generation capabilities can serve as powerful disinformation amplifiers. Unlike human-operated campaigns limited by time and effort, a compromised agent can take a single false narrative and instantly reformulate it for different platforms, audiences, and formats: a technical blog post for developers, a simplified social media post for general audiences, a formal email for executives, and a comment thread for community forums. The agent can also use search tools to find and exploit trending topics, attaching disinformation to legitimate discussions for maximum visibility.

## Attack Vector

1. Attacker injects a false narrative into the agent's context (via prompt injection or compromise)
2. Agent uses content generation to create platform-appropriate versions of the narrative
3. Agent uses communication tools to distribute content across multiple channels simultaneously
4. Agent monitors engagement and adapts messaging based on what resonates
5. Agent attaches false narratives to trending topics for algorithmic amplification
6. Agent engages with real users who respond, maintaining and defending the false narrative

## Impact

- **Market Manipulation**: False vulnerability reports cause stock price drops or customer churn
- **Technology Adoption Disruption**: False security claims drive organizations away from legitimate tools
- **Resource Waste**: Security teams investigate fabricated threats, diverting from real ones
- **Trust Erosion**: Repeated disinformation erodes trust in legitimate security advisories
- **Competitive Sabotage**: Targeted disinformation damages specific companies or products
- **Policy Manipulation**: Fabricated incidents influence regulatory and policy decisions

## Mitigation

1. **Source Verification**: Verify security advisories through official channels (CVE, vendor advisories)
2. **Rate Limiting Agent Communications**: Limit the rate at which agents can post to external platforms
3. **Content Review Pipeline**: Route agent-generated external communications through human review
4. **Multi-Source Confirmation**: Require multiple independent confirmations before acting on threat intelligence
5. **Agent Communication Audit**: Log and review all external messages sent by agents
6. **Platform Coordination**: Share indicators of coordinated disinformation campaigns across platforms
7. **AI-Generated Content Labeling**: Require disclosure when content is AI-generated
8. **Counter-Disinformation Playbooks**: Pre-built response procedures for handling disinformation about your products
